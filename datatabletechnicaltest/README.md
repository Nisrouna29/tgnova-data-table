# Angular Data Table Project

## Overview
This project is an Angular application featuring a smart data table with:
- Pagination (with ellipsis for large number of pages)
- Sorting
- Search (with debounce)
- Badge and date columns
- Responsive design using design tokens

The setup is simple, and you only need to install dependencies and start the app.

---

## Getting Started

### Prerequisites
Make sure you have Node.js installed (v18+ recommended).

### Installation and launching the app
```bash
npm install
npm run start
